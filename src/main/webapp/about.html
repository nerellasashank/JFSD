Problem Statement

The goal of this project is to develop a web-based career assessment tool that provides students with career assessments, personality tests, and skills evaluations. This tool will help students understand their strengths and preferences, guiding them toward suitable career options.

Modules

User (Student) Module:

User Registration: Allow students to create accounts to access the platform.
User Login: Enable students to log in to their accounts securely.
Career Assessments: Provide a variety of assessments and personality tests to help students evaluate their skills and interests.
View Results: Allow students to view their assessment results in an easy-to-understand format.
Explore Career Options: Provide recommendations and resources based on assessment results to help students explore potential career paths.

Admin Module:

Admin Login: Enable administrators to log in securely to the platform.
Manage Assessment Tools: Allow admins to create, edit, or delete assessments and tests available on the platform.
Update Test Data: Enable admins to update the questions and options for assessments as needed.
Provide Career Recommendations: Allow admins to develop and manage career recommendations based on assessment outcomes.
View User Activity: Provide functionality for admins to track user engagement and assessment completion rates.
